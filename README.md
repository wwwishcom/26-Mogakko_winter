# 26-Mogakko_winter
안소원의 26년 겨울, 백준 빙고 내기를 통한 BaekJoon 알고리즘 문제 풀이를 기록하였습니다.

## 목적
알고리즘 실력 향상 및 꾸준한 코딩 연습을 위해 빙고 게임 방식으로 백준 문제 풀이 내기를 진행한다.

## 진행 방식
문제 풀이를 꼭 블로그, 깃허브, 노션 등에 정리하여 단톡방에 제출할 것. 
제출하지 않으면 문제 풀이로 인정하지 않음

### 빙고판 구성
- 3x3 빙고판 (총 9칸)
- 각 칸은 자기 티어 등급 플마 5 문제로 채우기 (중복은 3개까지만)
- 예: 골드5면 [골5][골4][실3][실4][실5] 섞어서 만든 후 매주 월요일에 톡방에 어떤 식으로 진행할지 사진 보내기

### 빙고 완성
- 가로/세로/대각선 3줄 완성 시 종료
- 1등, 2등, 3등 순으로 순위 결정
- 1등 : 10점 / 2등 : 5점 / 3등 : 3점

### 보너스
- 일주일 안에 9문제 다 풀면 3점 추가
- 매주 꼴등은 1등,2등,3등의 빙고판을 정할 수 있다.
(물론 정하지 않고 자유에 맡길 수도 있음)

### 기간
- 매주 단위로 진행
- 매달 말일에 점수 정산
